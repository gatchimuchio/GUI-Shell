# First Run

GUI-Shell first run starts with Setup Doctor, not a terminal-first path.

Setup Doctor reports structured status for:

- Python availability
- Rust availability when native helper tests are needed
- Flutter availability when desktop UI analysis or launch is needed
- runtime connection readiness
- local permission prerequisites
- update policy contract
- audit storage contract
- recovery catalog contract
- adapter readiness

Installer and Setup Doctor state never grants authority and never approves permissions.
